An alert is a notification generated by a security tool when something suspicious is detected.

Examples:
- Multiple failed login attempts
- Malware detected by antivirus
- Unusual network traffic

Alert does NOT always mean an attack.

An incident is a confirmed security threat that can harm systems, data, or users.

Examples:
- Successful unauthorized access
- Malware infection confirmed
- Data breach

Incident requires immediate action.
